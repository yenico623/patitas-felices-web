# 🐾 Patitas Felices — Fundación de Rescate Animal

Maqueta web estática desarrollada para la Fundación Patitas Felices,
como proyecto académico de HTML y CSS nativo.

## 📋 Descripción
Página web responsive con múltiples secciones que presentan
la fundación, sus animales en adopción y canales de contacto.

## 🛠️ Tecnologías
- HTML5 semántico
- CSS3 nativo (Flexbox + Grid + @keyframes)
- Sin JavaScript
- Google Fonts: Fredoka One + Nunito

 ## ✨ Funcionalidades CSS
- Carrusel automático sin JavaScript (`@keyframes`)
- Hover effects en tarjetas y galería
